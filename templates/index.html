Forest Fire Weather Index (FWI) Prediction

Algerian Forest Fires Dataset

Data Set Information:

  • The dataset includes 244 instances from two regions of Algeria:
    • Bejaia region (northeast)
    • Sidi Bel-abbes region (northwest)
  • 122 instances for each region.
  • Data collected from June 2012 to September 2012.
  • The dataset includes 11 attributes and 1 output attribute (class).
  • The 244 instances are classified into Fire (138 instances) and Not Fire (106 instances) classes.

Attribute Information

  • Date: (DD/MM/YYYY) Day, month ('June' to 'September'), year (2012)
  • Temp: Temperature at noon (Â°C): 22 to 42
  • RH: Relative Humidity (%): 21 to 90
  • Ws: Wind speed (km/h): 6 to 29
  • Rain: Total rainfall of the day (mm): 0 to 16.8
  • FFMC: Fine Fuel Moisture Code index: 28.6 to 92.5
  • DMC: Duff Moisture Code index: 1.1 to 65.9
  • DC: Drought Code index: 7 to 220.4
  • ISI: Initial Spread Index: 0 to 18.5
  • FWI: Fire Weather Index: 0 to 31.1
